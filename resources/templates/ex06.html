th:object - 타임리프의 변수 선언 속성 - 적용된 태그 내부에서만 사용 가능 - div태그 안에서만 사용 가능한 지역변수이다. ex)
th:object="${session.user}" - 지역변수로 session에 저장된 user를 변수로 선언 *{name} - 변수로 저장된 user 객체의 getName메서드가 사용되었다. *{age} - 변수로 저장된 user 객체의 getAge메서드가 사용되었다.
