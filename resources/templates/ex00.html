타임리프의 표현식 #{} - 메세지 표현식 : messages.properties 파일에 정의된 속성을 사용 : 다국어 처리에 사용된다. ${} - 변수 표현식 Model 속성에 접근하는데 사용 (JSP에서 사용하는 표현언어와 동일) ${#} - 기본객체 표현식 lists 타임리프에 내장된 기본객체 중 lists를 사용 사용 예) ${#lists.size()} @{} - 링크 표현식 : URL을 생성하는데 사용 *{} - 선택변수 표현식 : 현재 객체의 내부변수(속성)에 접근하는데 사용 ~{} - 프래그먼트 표현식 : 다른 html을 포함시킬 때 사용 : JSP의 include와 같은 기능
